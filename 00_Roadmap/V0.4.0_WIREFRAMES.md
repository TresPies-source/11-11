## Overview

These wireframes define the complete UI structure for 11-11 v0.4.0 "UI Unification & Polish". They are based on research from Cursor, Windsurf, Replit, and Linear, adapted for 11-11's unique multi-agent architecture.

**Key Goals:**
- Unify fragmented UI into cohesive experience
- Make agent activity visible and transparent
- Support flexible, user-configurable layouts
- Optimize for primary workflows (thinking with AI, discovering knowledge, writing prompts)
- Maintain "calm" aesthetic while adding power-user features

---

## Design Principles

1. **Persistent Visibility:** Agent activity is always visible (no hidden pages)
2. **Flexible Layout:** User can resize/hide panels to match their workflow
3. **Hierarchical Navigation:** 3-tier sidebar (workspace > core nav > projects/recents)
4. **Progressive Disclosure:** Advanced features hidden until needed
5. **Multi-Agent Transparency:** Show all agents and their status at once
6. **Calm Aesthetics:** Minimal visual noise, focus on content

## Wireframe Summary

These wireframes define the complete UI structure for v0.4.0:

1. **Onboarding Wizard:** 3-step flow to introduce agents and workflows
2. **Dashboard:** Home screen with quick actions, agent status, and recent activity
3. **Sidebar:** 3-tier navigation (workspace, core nav, projects/recents)
4. **Workbench:** Flexible layout with Monaco editor and collapsible agent panel
5. **Agent Panel:** Multi-agent status, progress, cost tracking, and seed suggestions
6. **Librarian:** Full-page search with similarity scores and proactive suggestions
7. **Seeds:** Memory garden with Keep/Grow/Compost/Replant categories
