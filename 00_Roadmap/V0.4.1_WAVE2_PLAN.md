-   **Navigation:** Add clear navigation links to the dedicated Greenhouse and Commons pages.

### B. Greenhouse Page (`/librarian/greenhouse`)

**Problem:** The Greenhouse page is currently a blank slate.

**Solution:**
-   **Build from Scratch:** Create a new, full-featured Greenhouse page that allows users to:
