# Sprint 1 Screenshots

This directory contains visual documentation of the Sprint 1 UI Shell.

## Required Screenshots

1. **Full Layout (Sidebar Expanded)**
   - Shows complete Command Center with sidebar at ~300px width
   - File tree visible with all 6 planning directories
   - Header with workspace selector and sync status
   - Multi-Agent tab active

2. **Multi-Agent View (3+ Panels)**
   - Demonstrates responsive grid layout
   - Shows multiple chat panels with different agent personas
   - Illustrates panel controls (minimize, close)
   - NewSessionButton visible in bottom-right

3. **Header Detail**
   - Close-up of workspace selector
   - Sync status indicators (Google Drive + GitHub)
   - Mock user avatar/sign-in state

## Capturing Screenshots

To capture screenshots for this sprint:

1. Start the development server: `npm run dev`
2. Navigate to `http://localhost:3000`
3. Interact with the UI to create the desired state
4. Use your system's screenshot tool or browser DevTools
5. Save images to this directory with descriptive names:
   - `01-full-layout-sidebar-expanded.png`
   - `02-multi-agent-view-3-panels.png`
   - `03-header-detail.png`
   - `04-responsive-mobile.png` (optional)
   - `05-responsive-tablet.png` (optional)

## Notes

- Use PNG format for clarity
- Recommended resolution: 1920x1080 or higher
- Ensure dark mode is consistent across screenshots
- Capture both idle and active states where relevant
